Theory How it works

The (16,11) variant is an extended Hamming code. It uses 11 bits for data and 5 bits for parity:

  • ✔Bits 1, 2, 4, 8: Standard Hamming parity bits.
  • ✔Bit 0: The "Extended" parity bit (covers the entire block).
  • ✔Capability: Detects up to 2 errors, corrects 1.
